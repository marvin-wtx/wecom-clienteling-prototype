# Frozen Native WeCom Group-Send Frame

This is the generic V4.0 source for the **新建群发** frame. It contains only native-like structure and material treatments—no project copy, logo, person, commercial data, image, or customer information.

Copy the CSS and JavaScript into the matching marked blocks in a delivery. Bind recipients, message, selected image/mini-program material, picker, and completion result through the project-layer callbacks described in `references/native-wecom-broadcast-contract.md`. Run `scripts/check_native_wecom_broadcast.py case-directory --required` before delivery.
